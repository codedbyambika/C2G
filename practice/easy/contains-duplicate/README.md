# Contains Duplicate

## Difficulty: Easy

## Description
### Description
Given an array of integers, write a program to check if the array contains any duplicates.
Your program should print `true` if any value appears at least twice in the array, and print `false` if every element is distinct.

### Input Format
- The first line contains an integer **N** representing the number of elements in the array.
- The second line contains **N** space-separated integers representing the array elements.

### Output Format
Print `true` if duplicate exists, otherwise print `false`.

### Constraints
* $1 \\le N \\le 10^4$
* $-10^9 \\le \\text{nums}[i] \\le 10^9$

### Examples
**Example 1:**
Input:
```
4
1 2 3 1
```
Output:
```
true
```

**Example 2:**
Input:
```
4
1 2 3 4
```
Output:
```
false
```
